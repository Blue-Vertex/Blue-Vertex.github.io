Inferno


DevLog • Steam • Trailer • Twitter


Inferno is a simple 3d puzzle game set to release on steam in 2022.
The game is written from scratch in C++ without any external libraries.

Story

Inferno is based on Dante Alighieri's poem "Inferno" from his trilogy and magnum opus "The Divine Comedy". In Inferno Dante descends though the nine circles of Hell where he meets figures from his life in Florence as well as persons from Greek and Roman mythology.

Gameplay

The game starts out by Socoban style box-pushing mechanics. A design goal was to have each of the ~100 puzzles to be unique, with each level introducing some new idea. Each of the nine circles of hell comes with its own game mechanic.

  • Forest - pushing blocks
  • Circle I - Limbo - manipulating ladders
  • Circle II - Lust - locks and keys
  • Circle III - Gluttony - manipulating rolling souls
  • Circle IV - Greed - manipulating marching souls
  • Circle V - Wrath - manipulating light sources
  • Circle VI - Heresy - classical physical puzzles implemented in the game engine
  • Circle VII - Violence - switching between two characters to solve co-op puzzles
  • Circle VIII - Fraud - handling destructible blocks
  • Circle IX - Treachery - constructing logical circuits by combining all previous mechanics

Art and Litterature

Between levels quotes from "Inferno" are shown to the player. These come from the Longfellow translation, I chose this one because it is in the public domain and I like the sound of the old english language and the mood it brings. Coupled with the quotes are artwork. These are mostly Gustave Dore's famous wood engravings from the 18 centuary, colorized by me.


The game includes a dictionary explaining some uncommon words, it also gives brief intro on historical persons Dante meets.

Development

I started this project to learn c++ - my goal was to do the game from scratch without any external libraries. Development went relatively fast, I had a more or less complete game in 3 months: one month for the code, one month for making the levels, and one music, art, game menu, font, steam integration etc. Since then I've been polishing the game.
